Usage
=====

The ``ssc-cdi`` algorithm is called as ``sscCdi`` python module.

.. code-block:: python

    import sscCdi

    '''
    Set input data:

    input1 = ...
    input2 = ...
    
    Set dictionary parameters:

    dic = {}
    ...

    '''

    output = sscCdi.function_name(input1, input2, dic)
